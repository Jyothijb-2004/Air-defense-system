# Air-defense-system
designed and implemented an embedded C-based Air defense system prototype for real-time target detection and  tracking with automated threat alerts and response mechanism.
# overview
Developed Arduino-based system for real-time object detection and tracking.Implemented a automated laser turret alignment. Designed continuous surveillance using servo and stepper motor control.
# hardware used
-servo motor
-stepper motor
-ultrasonic sensor
-laser turret
-arduino uno
# software used
Arduino IDE
